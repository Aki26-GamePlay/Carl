# Carl

Carl  egy ChatBot-nak indult 2020. 12. 01-én.

Egyetlen Fejlesztője: Aki26 GamePlay

### Jelenlegi "Parancsok":
+ Beszélgetés
  + Hogy vagy?
  + Ki az apád?
  + Ki vagy te?
  + Te egy robot vagy?
  + Mennyi az idő?
  + Szeretsz?
  + Itt vagy?
  + Mi az igazi neved?
  + Mikor születtél?
  + Ki vagyok én?
+ Egyéb
  + Ötletet adnék.
    + Ötletet adhatsz a fejlesztőmnek.
  + Hibát jelentenék.
    + Hibát jelenthetsz a fejlesztőmnek.
  + Más a nevem.
    + Nevet változtathatsz.
